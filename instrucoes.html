Como Jogar

Seu objetivo é montar um computador o mais rápido possível

    • 1. Arraste a peça exibida até o slot vermelho correspondente
    • 2. Ao posicionar a peça corretamente, o slot ficará azul - solte para encaixar
    • 3. Cada peça encaixada automaticamente liberará a próxima componente
    • 4. Complete todas as conexões para ativar o botão POWER
    • 5. Quando o botão ficar VERDE, clique para ligar o sistema
    • 6. Registre seu nome para entrar no TOP 10 do placar
